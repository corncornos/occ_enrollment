## System Overview

The OCC Enrollment System is a comprehensive web-based platform designed to manage the complete student enrollment lifecycle from initial registration through graduation. The system supports multiple user roles, each with specific responsibilities and access levels.

### System Features

- **Student Registration & Application Management**
- **Document Upload & Verification**
- **Curriculum Management & Approval Workflow**
- **Enrollment Processing & Section Assignment**
- **Schedule Management**
- **Grade Entry & Viewing**
- **Certificate of Registration (COR) Generation**
- **Enrollment Reports & Analytics**
- **Chatbot FAQ System**
- **Adjustment Period Management**

### User Roles

1. **Student** - Primary users who register, apply, enroll, and view their academic information
2. **Admin/Registrar** - System administrators with full access to manage all aspects
3. **Admission Officer** - Handles applicant processing and document verification
4. **Program Head** - Manages curriculum for their respective programs
5. **Dean** - Reviews and approves curriculum submissions and enrollment reports
6. **Registrar Staff** - Assists with enrollment processes and administrative tasks

## Getting Started

### Accessing the System

1. Open your web browser and navigate to the OCC Enrollment System URL
2. You will see the login page

**Screenshot Placeholder:** `[INSERT LOGIN PAGE SCREENSHOT HERE]`

### Registration (For New Students)

1. Click on **"Register here"** link on the login page
2. Fill in all required information:
   - Personal Information (Name, Date of Birth, Contact Details)
   - Educational Background
   - Program Preference
   - Account Credentials (Email & Password)
3. Upload required documents (see Document Requirements section)
4. Submit the registration form
5. Wait for admission officer verification

**Screenshot Placeholder:** `[INSERT REGISTRATION FORM SCREENSHOT HERE]`

### Login Process

1. Enter your **Email Address** and **Password**
2. Click **"Sign In"** button
3. You will be redirected to your role-specific dashboard

**Note:** The system automatically detects your role and redirects you to the appropriate dashboard.

### Claiming an Existing Account

If you are an existing student but haven't created an account yet:

1. Click **"Claim your account"** link on the login page
2. Enter your student number and other required information
3. Set up your account credentials
4. Log in with your new credentials

## Troubleshooting

### Common Issues

#### Cannot Log In
- **Check credentials**: Ensure email and password are correct
- **Account status**: Verify your account is active
- **Contact support**: If issues persist, contact system administrator

#### Document Upload Fails
- **Check file size**: Ensure file is within size limits
- **Check file format**: Use supported formats (PDF, JPG, PNG)
- **Check internet connection**: Ensure stable connection

#### Cannot See Enrollment Option
- **Check enrollment period**: Verify enrollment is open
- **Check enrollment status**: Ensure you're eligible to enroll
- **Contact registrar**: If issues persist

#### Schedule Not Showing
- **Check enrollment status**: Ensure you're enrolled
- **Check section assignment**: Verify you're assigned to a section
- **Contact admin**: If schedule should be visible

#### COR Not Available
- **Check enrollment**: Ensure enrollment is complete
- **Check approval**: Verify enrollment is approved
- **Contact registrar**: Request COR generation

### Getting Help

- **Contact your Program Head** for curriculum-related questions
- **Contact Admission Office** for application and document issues
- **Contact Registrar** for enrollment and COR issues
- **Contact System Administrator** for technical problems

## Appendix

### Document Requirements

Students must upload:
1. **2x2 ID Pictures** (4 pieces)
2. **PSA Birth Certificate**
3. **Barangay Certificate of Residency**
4. **Voter's ID or Registration Stub**
5. **High School Diploma**
6. **SF10** (Senior High School Permanent Record)
7. **Form 138** (Report Card)
8. **Certificate of Good Moral Character**
9. **Transfer Credentials** (if applicable)

### Enrollment Statuses

- **Pending**: Application submitted, awaiting processing
- **Under Review**: Application being reviewed
- **Approved**: Application approved, ready for enrollment
- **Rejected**: Application rejected
- **Enrolled**: Successfully enrolled in the system

### Curriculum Statuses

- **Draft**: Curriculum being prepared
- **Submitted**: Submitted for review
- **Pending**: Awaiting approval
- **Approved**: Approved and active
- **Rejected**: Rejected, requires revision

### Keyboard Shortcuts

- **Ctrl + F**: Search on current page
- **Esc**: Close modals/dialogs
- **Enter**: Submit forms
