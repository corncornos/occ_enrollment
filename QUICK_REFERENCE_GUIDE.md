# OCC Enrollment System - Quick Reference Guide

## Quick Access by Role

### 👨‍🎓 Student
- **Dashboard**: View enrollment status, quick actions
- **Enroll**: Navigate to "Enroll for Next Semester"
- **View Schedule**: My Schedule → Schedule & Sections
- **View Grades**: Quick action button or menu
- **View COR**: My COR from menu
- **Upload Documents**: Documents → Upload Documents

### 👨‍💼 Admin/Registrar
- **Dashboard**: System overview and statistics
- **Manage Users**: User Management → View All Users
- **Approve Enrollments**: Enrollment Management → Pending Enrollments
- **Manage Curriculum**: Curriculum Management
- **Create Sections**: Section Management → Create Section
- **Enter Grades**: Grade Management → Enter Grades
- **Generate COR**: COR Management → Generate COR

### 📋 Admission Officer
- **Dashboard**: View pending applicants
- **Process Applicants**: Applicants → View Pending Applicants
- **Verify Documents**: Documents → Select Applicant
- **Assign Student Numbers**: Applicants → Assign Number
- **Pass to Registrar**: Applicants → Pass to Registrar
- **Set Enrollment Period**: Enrollment Control

### 📚 Program Head
- **Dashboard**: Curriculum statistics
- **Create Curriculum**: Curriculum Management → Create Submission
- **Bulk Upload**: Bulk Upload → Upload CSV
- **Review Enrollments**: Next Semester Enrollments
- **Submit Curriculum**: My Submissions → Submit

### 🎓 Dean
- **Dashboard**: Pending approvals overview
- **Approve Curriculum**: Curriculum Approvals → Review
- **View Reports**: Enrollment Reports → View Reports
- **Upload Curriculum**: Upload Curriculum → Bulk Upload

### 📝 Registrar Staff
- **Dashboard**: Pending enrollments
- **Process Enrollments**: View Pending Students
- **Generate COR**: Generate COR → Generate Certificate
- **Review Adjustments**: Review Adjustments

## Common Tasks

### For Students

**How to Enroll for Next Semester:**
1. Login → Dashboard
2. Click "Enroll for Next Semester" button
3. Select subjects
4. Choose section
5. Submit request
6. Wait for approval

**How to Upload Documents:**
1. Login → Documents → Upload Documents
2. Select document type
3. Choose file
4. Upload
5. Wait for verification

**How to View Schedule:**
1. Login → My Schedule → Schedule & Sections
2. View your class schedule with times and rooms

**How to Request Adjustment:**
1. Login → Dashboard
2. Click "Adjustment Period" (if open)
3. Add or drop subjects
4. Submit request

### For Admission Officers

**How to Process an Applicant:**
1. Login → Applicants
2. Click "View" on applicant
3. Go to Documents tab
4. Verify each document
5. Assign student number
6. Pass to Registrar

**How to Set Enrollment Period:**
1. Login → Enrollment Control
2. Set Academic Year and Semester
3. Set Opening and Closing Dates
4. Set Status to "Open"
5. Save

### For Program Heads

**How to Submit Curriculum:**
1. Login → Curriculum Management
2. Create new submission
3. Add subjects (manual or bulk upload)
4. Review and edit
5. Submit to Registrar

**How to Review Irregular Enrollments:**
1. Login → Next Semester Enrollments
2. View pending requests
3. Review subject selections
4. Approve or Reject

### For Admins

**How to Enroll a Student:**
1. Login → User Management → Pending Students
2. Select student
3. Assign to section
4. Approve enrollment
5. Generate COR

**How to Create a Section:**
1. Login → Section Management → Create Section
2. Select Program and Year Level
3. Enter section details
4. Set capacity and schedule
5. Save

**How to Enter Grades:**
1. Login → Grade Management → Enter Grades
2. Select student
3. Select subject
4. Enter grade
5. Save

## Status Indicators

### Enrollment Status
- 🟡 **Pending** - Awaiting processing
- 🔵 **Under Review** - Being reviewed
- 🟢 **Approved** - Approved and ready
- 🔴 **Rejected** - Rejected
- ✅ **Enrolled** - Successfully enrolled

### Document Status
- ⏳ **Pending** - Not yet verified
- ✅ **Verified** - Verified and accepted
- ❌ **Rejected** - Rejected (check reason)

### Curriculum Status
- 📝 **Draft** - Being prepared
- 📤 **Submitted** - Submitted for review
- ⏳ **Pending** - Awaiting approval
- ✅ **Approved** - Approved and active
- ❌ **Rejected** - Rejected

## Important Notes

⚠️ **Enrollment Period**: Check Enrollment Control for current enrollment dates

⚠️ **Document Verification**: Documents must be verified before enrollment

⚠️ **Section Assignment**: Students must be assigned to sections before enrollment

⚠️ **COR Generation**: COR is generated after enrollment approval

⚠️ **Adjustment Period**: Only available during designated periods
